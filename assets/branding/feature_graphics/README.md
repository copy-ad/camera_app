# TempCam Feature Graphics

These SVG files are prepared for store and promo use.

Files:

- `play_feature_graphic_primary.svg`
- `play_feature_graphic_vault.svg`
- `play_feature_graphic_camera.svg`

Recommended use:

- Google Play feature graphic: export one of these to `1024 x 500 PNG`
- Promo posts / web: keep SVG or export at higher resolution

If you want PNG output:

1. Open the SVG in a browser or design tool.
2. Export as `1024 x 500 PNG`.
3. Keep text inside safe margins so Play cropping does not cut it off.
