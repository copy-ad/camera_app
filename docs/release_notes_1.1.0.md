# TempCam 1.1.0 Release Notes

TempCam 1.1.0 focuses on privacy polish, user trust, and stronger day-to-day control.

## Highlights

- Added Panic Exit for quickly closing TempCam from key screens
- Added Stealth Notifications for more private lock-screen reminders
- Added Session Privacy Mode for instant relock when the app loses focus
- Added configurable Quick Lock Timeout options
- Added protected recent-app preview shielding
- Added trusted local vault history for:
  - Keep Forever exports
  - manual deletions
  - auto-deletions
- Improved gallery experience with:
  - real video thumbnails
  - filter tabs for photos and videos
  - multi-select delete
- Improved secure video playback inside TempCam
- Improved production notification behavior and Android notification support
- Updated legal links, store-readiness docs, and release hardening guidance
