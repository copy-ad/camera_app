# TempCam 1.4.0 Release Notes

TempCam 1.4.0 is the next internal testing build focused on a full UX redesign without changing the core feature set.

## Highlights

- Redesigned the main TempCam experience with a glassmorphism and depth-based visual system
- Improved motion, transitions, and micro-interactions across core flows
- Refreshed camera, vault, settings, lock, onboarding, splash, and paywall surfaces for a more premium and cohesive feel
- Preserved the existing feature set, privacy flow, and core behavior while improving usability and visual clarity

## Suggested Play Console Release Notes
