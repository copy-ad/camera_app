# TempCam 1.2.0 Release Notes

TempCam 1.2.0 adds faster private access and a safer path for bringing existing device media into the vault.

## Highlights

- Added Import to TempCam for bringing existing photos or videos into the vault
- Imported media now uses the same private storage and timer flow as in-app captures
- Added launcher quick actions for opening Camera or Vault faster
- Updated privacy documentation to cover imported media handling
- Refreshed product documentation for the latest privacy and access features
